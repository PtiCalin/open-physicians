# Data Dictionary

| Field | Type | Description |
| --- | --- | --- |
| physician_id | string | Stable unique identifier generated by `src/core/ids.py`. |
| full_name | string | Concatenated name normalized per locale rules. |
| specialty_code | string | Canonical specialty identifier sourced from `mappings/specialty_mapping.csv`. |
| license_status | string | Standardized license status derived from `mappings/license_status_mapping.csv`. |
| location_code | string | Region code aligned to ISO-3166 in `mappings/location_mapping.csv`. |
| source | string | Data source slug (e.g., `cmq`). |
| updated_at | datetime | Last observed update for the record in UTC. |
